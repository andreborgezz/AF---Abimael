# 🏆 copa do mundo 2026 — hub de noticias academico

## 📝 descricao do tema
um portal moderno e responsivo focado na cobertura ficticia da copa do mundo de 2026 na america do norte. o site reune informacoes divididas entre noticias de bastidores, inovacoes tecnologicas de campo e uma central de atendimento para torcedores do mundo todo.

## 🌐 paginas do projeto
- **home (`index.html`):** apresentacao do hub, noticias em destaque e o carrossel do bootstrap funcionando.
- **galeria (`galeria.html`):** mostra de novidades de lancamentos oficiais e fotos das sedes organizadas com o grid do bootstrap.
- **contato (`contato.html`):** area de atendimento ao usuario com mapa interativo do google via iframe e formulario validado.

## 🛠️ tecnologias utilizadas
- html5 semantico
- css3 (variaveis nativas, flexbox na estrutura e animacoes keyframes)
- bootstrap 5.3.3 (componentes de grid, navbar e carrossel)
- javascript (eventos de scroll, mouse e manipulacao do dom)
- versionamento via git e github

## ⚡ funcionalidades e interacoes com javascript
- **menu hamburguer responsivo:** navbar integrada do bootstrap adaptada para rodar em celulares.
- **botao inteligente de voltar ao topo:** criado e controlado de forma dinamica usando javascript puro.
- **validacao de formulario:** tratamento nativo com html5 e interatividade via script dando feedback para o usuario.
- **manipulacao do dom:** realce dos cards em tempo de execucao usando os eventos mouseenter e mouseleave do script.
